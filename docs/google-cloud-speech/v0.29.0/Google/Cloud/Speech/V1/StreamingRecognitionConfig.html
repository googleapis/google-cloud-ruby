#single_utterance ⇒ true, false

Returns Optional If +false+ or omitted, the recognizer will perform continuous recognition (continuing to wait for and process audio even if the user pauses speaking) until the client closes the input stream (gRPC API) or until the maximum time limit has been reached. May return multiple +StreamingRecognitionResult+s with the +is_final+ flag set to +true+.

If +true+, the recognizer will detect a single spoken utterance. When it detects that the user has paused or stopped speaking, it will return an +END_OF_SINGLE_UTTERANCE+ event and cease recognition. It will return no more than one +StreamingRecognitionResult+ with the +is_final+ flag set to +true+.
