Overview

Face

The results of face detection.

See #faces and #face.

Examples:

require "google/cloud/vision"

vision = Google::Cloud::Vision.new

image = vision.image "path/to/face.jpg"

face = image.face
face.confidence #=> 0.8616237640380859
