Overview

Vertex

A vertex in a set of bounding polygon vertices.

See Face::Bounds and Text.

Examples:

require "google/cloud/vision"

vision = Google::Cloud::Vision.new

image = vision.image "path/to/text.png"
text = image.text

text.bounds.count #=> 4
vertex = text.bounds.first
vertex.x #=> 1
vertex.y #=> 0
