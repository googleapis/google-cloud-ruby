#start_date ⇒ Google::Type::Date

Returns Optional.

Start date of the employment.

It can be a partial date (only year, or only year and month), but must be valid. Otherwise an error is thrown.

Examples: 2017, "month": 2, "day": 28 is valid. 2020, "month": 1, "date": 31 is valid. 2018, "month": 12 is valid (partial date). 2018 is valid (partial date). 2015, "day": 21 is not valid (month is missing but day is presented). 2018, "month": 13 is not valid (invalid month). 2017, "month": 1, "day": 32 is not valid (invalid day).

Returns:
